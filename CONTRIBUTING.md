# Installation

## Clone Repo

```bash
git clone --recurse-submodules git@github.com:Boshen/oxc.git
```

## Install Rust

```bash
# https://rustup.rs/
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

rustup show
```
